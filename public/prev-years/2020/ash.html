Submission Instructions

  1. Open Devpost.
  2. If you have not registered with Devpost, register now at that link
  3. Hit the green "Get Started" button on the right side of the screen
  4. Fill out the appropriate sections
  5. ALL team members have to be listed to be valid for prizes.
  6. In Project Description state IF you are doing a live demo
  7. In Project Description state which track (if any), and ONE other special category you are competing in
  8. Every demo will automatically be competing for Top Overall 3
  9. Upload your files and submit
